The IDE used for this project was InterliJ. The version of java used was 12.0.2. My TRE class is where the main method which is were the code is ran. TRE is located in project csc413-rezum-tankgame/src/tankgame/TRE. The a,d,w,s keys are the controls for the first player and the up, down, left, right are the control keys are for the second player. The first player shoot key is the space bar and the second player shoot key is the enter key. Once one player looses all the lives the game will end.
